# The Licensed Healer

A responsive website developed for **The Licensed Healer**, a licensed professional counselor offering counseling, LPC supervision, and holistic wellness services.

## Features

- Responsive design for desktop, tablet, and mobile
- Multi-page website
- Counseling and LPC supervision information
- Holistic wellness section
- Contact form
- Search engine optimization (SEO)
- Structured data (Schema.org)
- XML sitemap
- robots.txt
- Custom 404 page
- Accessible images with descriptive alt text

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery
